# 资产配置

## 概述

如果说选对一只基金或一只股票是“战术层面的胜利”，资产配置就是“战略层面的布局”。大量的学术研究与实践数据反复印证同一个结论：一个投资组合长期收益波动的90%以上，由资产配置决策决定，而非择时交易或个券选择。资产配置的本质不是预测哪类资产明年涨得最好——那是一种妄念——而是通过将资金科学地分散到不同类别、不同地域、不同风险收益特征的资产中，构建一个在任何经济环境下都不会彻底崩溃的组合结构。本主题从资产类别的本质理解、战略配置与战术调整的框架、再平衡纪律、生命周期适应性到常见的行为误区，系统阐述将资产配置从理论概念转化为个人实操体系的完整路径。

## 核心要点

### 1. 资产类别的底层图谱：理解你在买什么

资产配置的前提是对可选资产的风险收益特征有清晰认知。在个人投资者的可操作范围内，核心资产类别如下：

- **现金及等价物**：银行活期存款、货币基金、超短债基金。特征是无本金波动、即时流动性极高、收益率处于各类资产最低水平。在组合中的角色是流动性储备和短期资金的临时栖息地，而非增值工具。
- **固定收益类资产**：国债、地方政府债、高等级企业债、债券基金。特征是预期收益高于现金、净值存在温和波动、与股票市场通常呈低相关或负相关关系。在组合中的核心角色是为整体波动提供稳定锚，并在股票市场下行时充当“弹药库”和缓冲垫。
- **权益类资产**：股票、股票基金、指数基金。特征是长期预期收益率在各大类中最高、但短期波动幅度也最大，持有过程中回撤20%-50%是正常的历史波动区间。在组合中的角色是长期资本增值的核心引擎，需要以5年以上的持有周期与之匹配。
- **另类资产**：黄金、REITs（房地产投资信托）、大宗商品基金等。特征是与传统股债资产的相关性较低，能在特定经济环境下（如高通胀时期）提供差异化回报。在组合中的角色是分散器，配置比例通常控制在5%-15%。
- **人力资本**：容易被忽略但往往是最重要的“隐性资产”——你未来职业生涯中能创造的劳动收入折现值。年轻人的主要财富是人力资本而非金融资本，这一认知会深刻影响早期资产配置的风险承受度。

### 2. 战略资产配置：你的长期基准线

战略资产配置是为资产组合设定一个长期的、不随市场短期波动轻易改变的目标比例，它构成你全部投资行为的锚点：

- **确定股债配比**：这是战略配置中最核心的一步。经典的参考框架包括“100减去年龄”法则（100减去当前年龄所得数字作为权益类资产的目标占比），或更保守的“120减去年龄”作为权益比例的参考上限。但这一规则需要结合个人收入稳定性、财务目标期限和主观风险承受能力进行个性化修正。
- **确定地域分配**：在权益类资产的内部，进一步分配于国内与国际市场，避免将全部财富暴露在单一国家的经济周期与政策风险之下。对于中国投资者而言，宽基指数已覆盖国内主要上市公司，辅以全球市场指数基金可实现跨市场分散。
- **资产配置的政策声明**：模仿机构投资者的做法，为自己写一份简单的《投资政策声明书》，内容包括：投资目标、可接受的最大回撤幅度、战略资产配置比例、再平衡频率、以及什么情况下才允许偏离战略配置。这份文书的核心功能，是让你在极端市场情绪中有一份自己对自己做出的书面承诺可以回望。

### 3. 再平衡：用规则代替情绪

市场涨跌会持续地改变组合中各类资产的实际占比，牛市会让权益比例被动膨胀，熊市则反之。再平衡就是定期将资产比例拉回战略配置线的操作：

- **再平衡的实质**：在客观上实现了“高卖低买”——卖出一部分涨得过高的资产类别，买入一部分跌得过低的资产类别。这是为数不多的、被反复证明能贡献额外收益的纪律性操作。
- **触发机制的选择**：再平衡可按固定时间间隔（如每年一次或每半年一次）执行，也可按偏离阈值（如实际比例偏离目标线5个百分点时触发）执行。对于个人投资者，年度再平衡已足够有效，过于频繁的调整会增加交易成本与税负摩擦。
- **执行再平衡的心理逆人性**：真正的困难在于，再平衡要求你在牛市卖出还在涨的资产，在熊市买入还在跌的资产——这在直觉上恰恰与舒适感背道而驰。正因为如此，再平衡只有被固化为机械执行的规则才可能被坚持，依赖临场判断则大概率失败。

### 4. 生命周期资产配置：不同的阶段，不同的任务

资产配置方案必须随人生阶段同步演化，不存在一个从25岁用到65岁不变的配置比例：

- **积累期（20-35岁）**：人力资本充裕，金融资本尚少，最大的优势是时间——有足够长的投资期限来熨平权益资产的短期波动。此阶段可配置较高比例的权益类资产（如70%-90%），并应优先通过定投方式逐步建立组合。
- **巩固期（35-50岁）**：金融资本已具规模，但家庭经济责任也达到顶峰。组合的首要任务开始从“最大化增值”向“增值与保护并重”过渡，权益比例可适度下调（如60%-75%），并增配债券类资产以降低组合波动，匹配更稳健的家庭现金流需求。
- **过渡期（50岁至退休前）**：收入峰值通常出现在此阶段初期，随着退休节点的逼近，组合需要开启一条“向下滑行”的路径——逐渐降低权益仓位，锁定部分浮盈，将组合的重心从增长转向资本保值。经典的“目标日期基金”模式就是在这一阶段按照预设轨迹逐年调降权益比例。
- **消耗期（退休后）**：主动收入停止，开始依赖投资组合提供现金流。此时的核心矛盾从“如何增长”转为“如何在有生之年不耗尽资产”。权益比例通常降至30%-50%，固定收益与现金类资产占主导。同时需设定安全提取率（年提取金额÷组合总市值，通常建议不超过4%），并对提取节奏进行动态调整。

### 5. 行为陷阱：资产配置中最常见的错误动作

资产配置的执行力远比理论理解更为稀缺，以下几个行为偏差是导致配置失效的常见原因：

- **近期回报外推**：看到某类资产过去一两年表现优异就大幅超配，看到某类资产连续下跌就清仓回避。这种行为恰恰是在买入已经变贵的资产、卖出已经变便宜的资产，与投资获利的底层逻辑反向操作。
- **频繁战术调整**：试图通过不断地微调配置比例来追逐短期热点或躲避短期波动。每一次调仓都意味着一次方向性预测，而持续正确预测市场几乎是不可能的。战略配置的价值恰恰在于屏蔽这种无效的短期决策噪音。
- **账户割裂看待**：将不同账户——银行理财、证券账户、公积金、养老保险个人账户——割裂看待，而非作为统一投资组合的整体来考虑。这会导致整体配置比例在不知不觉中偏离目标。资产配置必须将所有可投资资产合并到一个全局视角下进行审视。
- **对现金的错误偏见**：在牛市中极度厌恶持有任何现金，在熊市中又不敢将任何资金投入市场。现金在资产配置中始终有合法地位——它是你的流动性储备、再平衡弹药以及应对突发状况的保障。现金比例应基于流动性需求来设定，而非基于市场情绪。

### 6. 极端风险下的资产配置韧性

一个真正稳健的资产配置，应该在设计阶段就思考过极端情境：

- **相关性陷阱**：在正常市场环境下，不同资产之间的低相关性提供了分散化的好处。但在极端系统性危机中，风险资产之间的相关性会急剧上升——资金会同时抛售股票、信用债甚至部分避险资产。真正的分散化需要跨越资产类别、地域与货币，现金与国债是系统性危机中为数不多的有效对冲工具。
- **通胀情境**：股债皆受通胀侵蚀，尤其是长期固定利率债券在高通胀环境中损失惨重。在配置中保持一定比例的权益资产（优质企业能将通胀向下游转移）、以及少量抗通胀的另类资产（如黄金、通胀挂钩债券），是对冲这一风险的结构性安排。
- **长期停滞情境**：如果面临类似日本“失去的三十年”的长周期低增长低通胀环境，过于集中的本土权益配置会遭遇巨大挑战。全球分散化配置（通过QDII基金或跨境ETF投资于不同经济体）的价值在这一情境下被放大——国运不同步，是配置全球化的最根本理由。
